# 📝 Flask To-Do Web Application

A simple and responsive To-Do web application built using Flask and Bootstrap. This project demonstrates basic CRUD operations and backend development concepts.

## 🚀 Features
- Add new tasks with title and description
- Update existing tasks
- Delete tasks
- Display tasks in a clean, numbered list
- Responsive UI using Bootstrap

## 🛠️ Technologies Used
- Python
- Flask
- SQLite
- SQLAlchemy
- HTML, CSS (Bootstrap)
