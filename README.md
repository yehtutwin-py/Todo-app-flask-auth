📝 Flask To-Do Web Application with Authentication

A secure and responsive To-Do web application built using Flask and Bootstrap. This project demonstrates user authentication along with full CRUD operations and backend development concepts.

🚀 Features
User registration and login system 🔐
Secure authentication with session handling
Add new tasks with title and description
Update existing tasks
Delete tasks
Each user can manage their own tasks
Display tasks in a clean, numbered list
Responsive UI using Bootstrap

🛠️ Technologies Used
Python
Flask
SQLite
SQLAlchemy
Flask-Login (or session-based auth)
HTML, CSS (Bootstrap)
